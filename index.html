Magnifying Glass Tool

Double-click anywhere to activate/deactivate the magnifying glass. It will show the text with inverted colors.

Installation Instructions

  1. Save this page as a bookmark by pressing Ctrl+D (Windows/Linux) or Cmd+D (Mac).
  2. You can also download all files and run them locally on any computer.
  3. On mobile devices, use two fingers to tap and activate the magnifier.
  4. Double-click anywhere to deactivate the magnifier.
